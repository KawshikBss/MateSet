## General_Info
MateSet is a social media platform where users can follow eachother, upload posts and chat.

## Technologies
* Python: 3.9.2
* Flask: 1.1.2
* Flask-sql-alchemy
* HTML: 5
* CSS: 3
* Javascript
* Jinja

## SetUp
To run the project, install the following modules using pip:
'''
$ pip install Flask
$ pip install SQLALchemy
'''
